В папке detector лежит код детекции дронов, работающий на rapsberry Pi и отправляющий информацию на сервер.
В папке drone_detector_web находится код для сервера.
В папке old_detector расположен старый код детекции дронов, в который включена спектрограмма для возможности визуальной проверки алгоритма.
Чтобы запустить сервер нужно в PowerShell открыть файл server.py.
На raspberry Pi код запускается при включении.
